Tentang saya

Perjalanan koding mengoding

Nama saya Muhamad Diaz Ramdani, saya lahir di Sukabumi, 25 Januari 1998, saya adalah anak pertama dari 3 bersaudara, hobi saya adalah bermain Video Games, Saya bersekolah dasar di SDN 4 Parakansalak, lalu saya melanjutkan Sekolah menengah pertama di SMP Yasidik Parakansalak, dan saya melanjutkan Sekolah menengah atas di SMAN 1 Parakansalah, Saya mulai tertarik pada dunia koding di mulai dari kelas 2 SMA, entah mengapa saya mulai sadar saat itu tentang bagaimana caranya membuat game, dari situ saya coba cari berbagai informasi tentang cara membuat game dan akhirnya saya bertemu dengan yang namanya koding mengoding :), bahasa pemrograman yang paling pertama saya pelajari adalah Java, saat itu saya masih belajar ngoding dengan cara hapal sintaks dan memang awalnya mudah dengan cara tersebut tapi lama kelamaan kok jadi semakin pusing dan ruwet dari situlah saya mulai mengalihkan impian saya dari membuat game menjadi seorang web developer saya beralih dari Java ke HTML, saya mulai pelajari HTML dari konsep nya dengan meninggalkan metode hapal sintaks yang dulu saya terapkan saat belajar Java, dari situ saya mulai lebih menyukai HTML ketimbang Java karena lebih mudah dipelajari dan saya sekarang meninggalkan Java sepenuhnya dan ingin menjadi seorang Full Stack Web Developer, Sekian perjalanan sederhana saya dalam dunia koding mengoding :).
